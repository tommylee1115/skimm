<p align="center"><i>Read slowly. 천천히 읽다.</i></p>

<p align="center">
  A quiet markdown reader for people who want to actually finish what they open.
</p>

<br />

<table align="center">
  <tr>
    <td align="center" width="33%">
      <h3>Visual chunking</h3>
      Paragraphs break into readable beats so your eyes never get lost mid-sentence.
    </td>
    <td align="center" width="33%">
      <h3>Read aloud</h3>
      Alt-click any word to start narration there. Natural voices, not robotic.
    </td>
    <td align="center" width="33%">
      <h3>Vocab in context</h3>
      Definitions surface inline as you read — no tab-switching, no flow break.
    </td>
  </tr>
</table>

<br />

## Quickstart

```bash
npm install
npm run dev
```

## Build

```bash
npm run package    # Windows installer
npm run ship       # tag, build, release
```
